# Booking/Travel Website Testing - PHPTravels

Automated E2E testing for travel booking website demonstrating complex search, filtering, and multi-step booking flows.

## Test Coverage

| Module | Tests |
|--------|-------|
| Hotel Search | 5 |
| Search Results | 4 |
| Booking Flow | 4 |
| Form Validation | 3 |

## Features Tested

- Hotel search with destination, dates, guests
- Search results filtering and sorting
- Hotel detail page navigation
- Multi-step booking flow
- Form validation
- Price calculation verification

## Run Tests

```bash
# From root directory
npm run test:booking

# Or from this directory
npx playwright test
```
